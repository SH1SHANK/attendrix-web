# Introduction

Welcome to the **Attendrix** documentation. This guide will help you understand how to use Attendrix to track your attendance, calculate safe cuts, and never worry about the 75% attendance threshold again.

## What is Attendrix?

Attendrix is a modern attendance tracking platform designed specifically for **NIT Calicut students**. Unlike traditional overall-percentage calculators, Attendrix tracks attendance **per subject**, giving you precise insights into exactly how many classes you can safely skip.

> ⚡ **Pro Tip:** Connect your Telegram to get real-time attendance alerts via our Lumen AI assistant.

## Key Features

- **Subject-wise Tracking** — Track attendance for each subject individually
- **Safe Cut Calculator** — Know exactly how many classes you can skip per subject
- **Lumen AI Assistant** — Get intelligent recommendations via Telegram
- **Amplix Gamification** — Earn streaks, badges, and compete on the leaderboard
- **Real-time Sync** — Your data is synced instantly across all devices

## Getting Started

### Prerequisites

Before you begin, ensure you have:

1. A valid NITC student email (`@nitc.ac.in`)
2. Telegram app installed (for Lumen AI)

### Quick Setup

Getting started takes under 2 minutes:

```text
# Step 1: Open the Telegram bot
https://t.me/HeyLumenBot

# Step 2: Send /start to begin onboarding

# Step 3: Select your Batch and Semester
```

That's it! Attendrix will auto-fetch your timetable and electives from the system.

## Architecture Overview

Attendrix is built on a modern tech stack:

| Component      | Technology                |
| -------------- | ------------------------- |
| Frontend       | Next.js 15                |
| Authentication | Firebase Auth             |
| Database       | Supabase (PostgreSQL)     |
| AI Assistant   | Telegram Bot API + Gemini |
