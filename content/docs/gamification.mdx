# The Amplix System

**Amplix** is the experience point (XP) currency of Attendrix. Users earn Amplix by:

1.  **Attending Classes**: Base reward per class.
2.  **Completing Challenges**: Large bonuses for Weekly/Monthly goals.
3.  **Streaks**: Multipliers for consecutive attendance.

# Mage Ranks

As users gain Amplix, they ascend through 12 unique Mage Ranks. This status is displayed on the Profile ID Card.

<Callout type="tip" title="Design Note">
  The system uses a non-linear progression curve. Early levels are fast to
  encourage engagement, while higher levels require significant consistency
  (Semesters of attendance).
</Callout>
