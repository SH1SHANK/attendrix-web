# Attendrix Feature Summary

Last reviewed: 2026-02-05

## Dashboard

- Today schedule and upcoming classes.
- Attendance summary and progress indicators.
- Installable PWA banner.

## Attendance

- Check-in and mark-absent actions.
- Bulk check-in for multiple classes.
- Past classes list with filters.
- Summary and challenge evaluation flows.

## Classes

- Date-based schedule view.
- Attendance actions embedded in class list.

## Tasks

- Read-only tasks and exams list.
- Tabs for assignments and exams.

## Profile

- Course editing and enrollment updates.
- CSV, Markdown, and branded PDF exports.
- Calendar sync section with help guidance.
- Resync user records tool.
- Deactivate and delete account flows.
- About and Support actions.

## Support

- Bug and feature request forms.
- Moderation and batch access requests.
- GitHub issue creation with image attachments.

## PWA

- Manifest and service worker for installable experience.
- Install prompt with "Don't show again" persistence.
