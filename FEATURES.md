# Attendrix Feature Summary

Last reviewed: 2026-02-05

## Dashboard

- Today schedule and upcoming classes.
- Attendance summary and progress indicators.
- Installable PWA banner.

## Attendance

- Check-in and mark-absent actions.
- Bulk check-in for multiple classes.
- Past classes list with filters.
- Summary and challenge evaluation flows.

## Classes

- Date-based schedule view.
- Attendance actions embedded in class list.

## Tasks

- Read-only tasks and exams list.
- Tabs for assignments and exams.

## Academic Resources

- Academic repository per enrolled course.
- Google Drive-backed folder/file browsing (read-only).
- Favorites, recently opened, and tagged sections.
- Drag-and-drop section ordering persisted in user preferences.
- Slash-based tag search with pinned tags and tag suggestions.
- Inline quick actions for star/unstar and offline access.
- Command Center (âŒ˜/Ctrl+K) and keyboard shortcuts for power users.
- Offline-only page (`/resources/offline`) with course/tag/recent grouping.
- Dedicated study materials settings page (`/resources/settings`) for cache and offline controls.
- Offline cache with storage usage indicator, cache limit slider, cleanup assistant, and manual clear.
- Offline storage mode: web cache (default) or dedicated device folder with permission + fallback.
- Focus Mode to hide actions and filters for distraction-free browsing.
- Personal repository placeholder (coming soon state).

## Profile

- Course editing and enrollment updates.
- CSV, Markdown, and branded PDF exports.
- Calendar sync section with help guidance.
- Resync user records tool.
- Deactivate and delete account flows.
- About and Support actions.

## Support

- Bug and feature request forms.
- Moderation and batch access requests.
- GitHub issue creation with image attachments.

## PWA

- Manifest and service worker for installable experience.
- Install prompt with "Don't show again" persistence.
